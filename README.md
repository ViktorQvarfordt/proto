# Proto

This proto-repo exists to bootstrap new projects.

It's a fullstack monorepo with a server and client.

- TypeScript
- React
- Vite
- React Router
- tRPC
- Zod
- Bun
- Biome
- Styled Components
- Jotai

## Dev

**Install dependencies**

```
cd ts
bun i
```

**Run**

```
# From repo root
./dev
```
